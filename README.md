# Java-DSA-Basics

## Description
This repository contains beginner-level Java DSA programs focused on numbers and basic patterns. 
It includes programs using loops ,nested loops, conditionals, and variables (data types).

## Topics Covered
- Basic number problems: Reverse, Prime, Factorial
- Loops and Conditionals (used in programs)
- Patterns using nested loops
